## Features

- **Shake Counter**: Подсчитывает каждое резкое встряску на основе данных о движении устройства.
- **Shake Animation**:Обеспечивает визуальную подсказку (например, цветную вспышку или встряску) для каждого обнаруженного встряски.
- **Error Handling**:Отображает сообщение об ошибке, если устройство не поддерживает датчики движения или доступ к датчикам запрещен.

## Technologies Used

- **React**: Фронтенд-библиотека для создания пользовательских интерфейсов.
- **TailwindCSS**: Утилита CSS-фреймворка для стилизации.
- **Sensors API**:Для доступа к данным о движении устройства.

### Логика обнаружения дрожания
Это приложение использует API DeviceMotionEvent для отслеживания изменений ускорения по осям X, Y или Z. «Встряска» определяется как быстрое изменение скорости более 15 м/с² по любой из этих осей, что указывает на сильное встряхивание. Каждый раз, когда обнаруживается встряска, счетчик увеличивается на 1, а анимация реагирует, предоставляя визуальную обратную связь.

### Использование
Встряхните устройство: счетчик встряхивания будет увеличиваться при каждом обнаружении встряхивания, а на это событие будет реагировать анимация.
Сброс счетчика: нажмите кнопку «Сброс счетчика», чтобы сбросить счетчик встряхиваний до нуля.
Обработка ошибок: если датчики движения не поддерживаются или доступ запрещен, появится сообщение об ошибке.

### Поиск неисправностей
Разрешения: на устройствах iOS обязательно включите «Доступ к движению и ориентации» в разделе «Настройки» > «Safari».
Тестирование на iOS. Обратите внимание, что Chrome на iOS также соответствует настройкам разрешения движения Safari из-за ограничений iOS WebKit.
